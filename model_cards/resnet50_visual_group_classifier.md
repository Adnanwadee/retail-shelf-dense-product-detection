# ResNet-50 Visual Product-Group Classifier Model Card

## Model

ResNet-50 classifier for visual product-group pseudo-labels.

## Purpose

Classify cropped product instances into visual product groups generated by unsupervised clustering.

## Task Type

Image Classification.

## Important Labeling Note

The 91 classes are not manually verified commercial SKU labels.

They are visual product-group pseudo-labels generated through:

```text
Product crops
  → DINOv2 visual embeddings
  → Mini-Batch K-Means clustering
  → 91 visual product groups
```

## Intended Use

This classifier is an optional second-stage component for dashboard grouping and inventory-style analytics.

## Not Intended For

- Verified SKU recognition
- Commercial billing
- Point-of-sale automation
- Product identity verification without a real SKU database

## Deployment Notes

If the ResNet-50 weights are not available, the Streamlit app can still run in detection-only mode.

## Limitations

- Classification quality depends on detection crop quality.
- Pseudo-labels may group products by visual similarity rather than exact identity.
- Real SKU-level deployment requires verified labels and a retail product database.
